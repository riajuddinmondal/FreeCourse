Introduction of Python

Python is a general-purpose interpreted, dynamic, object-oriented, and high-level programming language. This tutorial is designed for beginner as well as professional programmers who need to learn Python programming language from scratch. You should have a basic understanding of Computer Programming terminologies in order to learn Python. If you know any other programming language then it will be beneficial for you.

Python 2.x vs. Python 3.x

  • print "hello" (Python 2.x). print("hello") (Python 3.x)
  • Python 2.0 was first released in 2000. Python 3.0 was first released in 2008. Python 2.0 was first released in 2000. Python 3.0 was first released in 2008.

  • In Python 3.x syntax is simpler and easily understandable. Syntax of Python 2.x was comparatively difficult to understand

  • If you write code in Python 2.x a number without any digits after the decimal point, it rounds your calculation down to the floor. But Python 3.x, the expression 7 / 2 will return the expected result of 3.5
  • Recently many developers are creating libraries which you can use only with Python 3.x , older libraries created for Python 2.x is not forward-compatible
